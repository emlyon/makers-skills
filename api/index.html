Tools

IFTTT is a free online application for binding various services together: when an event is triggered in one service, IFTTT triggers an action in the other service. The name "IFTTT" comes from If This Then That, a formula well known to programmers.

Keep in mind that IFTTT simplifies the use of APIs, which are actually more complex and require coding skills to be used properly.

Going further

Definitions

An API is made available to developers to connect different IT tools to each other without having to understand or modify the source code.

Depending on the nature of the software, an API can be more or less open. Indeed, a site like Facebook or Instagram has every interest in opening its API as much as possible to facilitate the connection with other sites and generate a lot of traffic. On the other hand, Health Insurance cannot afford such an opening because it can compromise the security of users' personal data.

A REST API is a way to communicate data using HTTP methods such as POST, GET, PUT, DELETE and so on. It describes the actions to manage data in a semantical way.
