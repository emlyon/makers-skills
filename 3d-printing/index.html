Tools

Thingiverse is a platform gathering a large community of makers, where you can find and share 2D and 3D files.

The Zortrax M200 is a 3D printer you may find at the makers' lab, in any emlyon campus: this machine is at your disposal! At the end of this course, do not hesitate to use this amazing tool for your own projects.

Z-suite is Zortrax Slicer. You'll configure your print and send it to the SD card through this software.

Project

To do it yourself, you can download and 3D print an emlyon marvin test from Thingiverse. You will always find Z-ULTRAT material loaded on the machine at makers' lab.

We strongly recommend to choose 0.19mm as layer thickness parameter in Z-SUITE for this project. You also have some parameters recommendations in the intro to 3D printing video. Set 0° on support parameter when you print your marvin test. 0° settings remove all supports. Supports take time and material but could be necessary for some 3D prints. You can also read the complete documentation of Z-SUITE parameters.

Going further

Definitions

3D means three-dimensional : The three dimensions are often called length (or depth), width and height.

.stl is the most common file format for 3D printing. A STL file only contains the shape of your model, not colors. It can be opened with 3D softwares only, like Tinkercad or Z-suite.

A slicer is the name given to the softwares generating a machine code from a 3D file (like .stl). Basically it will transform your model into layers for the 3D printers.

There are lots of different 3D printing technologies. Discover the main ones :

  • FDM is the most common and affordable method: it uses thermoplastic filament which is heated and extruded layer by layer. The Zortrax M200 in the makers'labs are FDM machines.
  • SLA is a method using an UV laser beam to cure a liquid resin. It's more occurate than FDM and used for very detailed projects. Watch a timelapse video of the Formlabs 3D printer.
  • SLS is an industrial process, using a laser to sinter powder material.
